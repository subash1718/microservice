# Microservices Project

This repository contains a microservices-based application developed for the Microservices Architecture and Continuous Build & Delivery modules.

## Planned Components
- Config Server
- Eureka Discovery Server
- API Gateway
- Order Service
- Product Service
- CI/CD Pipeline (Jenkins)

## Purpose
The project demonstrates:
- Service discovery
- Centralised configuration
- REST-based microservices
- Security using JWT
- CI/CD automation
